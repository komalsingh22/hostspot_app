# Hostspot App

A Flutter application for sharing and experiencing multimedia content.

## Features

- Browse and select different "experiences".
- Answer questions related to experiences.
- Record and play back audio responses.
- Record and play back video responses.

## Technologies Used

This project is built with Flutter and Dart, and utilizes the following packages:

- **State Management:** [flutter_riverpod](https://pub.dev/packages/flutter_riverpod)
- **Networking:** [dio](https://pub.dev/packages/dio)
- **Image Loading:** [cached_network_image](https://pub.dev/packages/cached_network_image)
- **Audio:**
  - [record](https://pub.dev/packages/record)
  - [audioplayers](https://pub.dev/packages/audioplayers)
  - [flutter_sound](https://pub.dev/packages/flutter_sound)
- **Video:**
  - [camera](https://pub.dev/packages/camera)
  - [video_player](https://pub.dev/packages/video_player)
  - [chewie](https://pub.dev/packages/chewie)
- **Permissions:** [permission_handler](https://pub.dev/packages/permission_handler)
- **File System:** [path_provider](https://pub.dev/packages/path_provider)
- **UI:** [flutter_svg](https://pub.dev/packages/flutter_svg)
